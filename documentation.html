OECD Pharma Visualization – Design Documentation

Project Overview

This project presents an interactive D3.js line chart to visualize pharmaceutical expenditure data from 2013 to 2023 across OECD countries. Users can explore trends in per-capita pharmaceutical sales by country and drug type using dropdown menus and dynamic transitions.

Data Source

This dataset was obtained from the OECD Health Data Explorer: OECD Pharmaceutical Market Dataset (2013–2023)

Design Process

Two initial hand-drawn sketches helped determine layout, interaction methods, and chart type before coding.

Sketch 1: Initial bar chart with dropdown layout and tooltip concept.

Sketch 2: Revised sketch with a line chart to show trends over time.

Why a Line Chart?

  • Line charts show continuous data and trends more clearly than bars.
  • They highlight year-over-year changes without overwhelming the user.
  • The smooth transitions and clean design reduce visual noise and improve readability.

Visual Encoding Choices

VariableEncodingReason
YearX-axisChronological left-to-right makes trends easy to follow.
Pharma ValueY-axisHeight encodes magnitude of per-capita sales.
CountryDropdown FilterAllows focused analysis per country.
Pharmaceutical TypeDropdown + ColorUsers can isolate or aggregate types.
Value DetailTooltipReduces clutter, reveals exact info on hover.

Layout Decisions

Dropdowns are positioned above the chart for easy access. The title provides context, and a tooltip follows the cursor for instant feedback. Axes and lines are clearly labeled and animated to draw attention to trends.
